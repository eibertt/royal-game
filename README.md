# 🤖 TITAN B - Cyber-Futuristic Mining Game
**High-End Telegram Mini App | Built with Three.js & TON Connect**

Este proyecto es un juego de minería híbrida (Manual + Automática) diseñado para el ecosistema de Telegram.

## 🚀 Características
- **Gráficos 3D PBR:** Renderizado de alta fidelidad para el Robot B.
- **Economía Web3:** Integración nativa con la red TON.
- **Sistema Modular:** Evolución de armaduras cerámicas blancas.
- **Despliegue Continuo:** Optimizado para Vercel.

## 🛠️ Stack Tecnológico
- **Frontend:** HTML5, CSS3 (Orbitron Font), JavaScript.
- **Engine:** Three.js (WebGL).
- **Backend:** Supabase (PostgreSQL).
- **Blockchain:** TON SDK.
